# 📝 FastAPI Todo API

A simple FastAPI-based Todo application with SQLite + Docker support.

---

## 🚀 Features

- ✅ REST API with FastAPI
- 🗃️ SQLite database using SQLModel
- 🐳 Dockerized for easy deployment
- 🔄 GitHub Actions CI for linting and build test

---

## 📦 Tech Stack

- FastAPI
- SQLModel
- SQLite
- Docker
- GitHub Actions

---
